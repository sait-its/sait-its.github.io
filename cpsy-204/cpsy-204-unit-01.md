### What is Linux?

- **Definition**: Linux is a free, open-source operating system kernel that serves as the foundation for various distributions.
- **Origins**: Developed by **Linus Torvalds** in 1991 as a personal project; it has since become the most widely used kernel in the world.
- **Key Characteristics**:
  - **Stability**: Can run for years without rebooting.
  - **Security**: Built-in permissions and a robust user privilege model.
  - **Flexibility**: Can be stripped down for a tiny IoT device or scaled up for a massive supercomputer.

- **The Ecosystem**: It is maintained by a global community of developers and the Linux Foundation.
